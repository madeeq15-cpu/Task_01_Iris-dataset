# Task 1: Exploring and Visualizing the Iris Dataset

## Objective

The objective of this task is to understand how to load, inspect, summarize, and visualize a dataset using Python libraries such as Pandas, Matplotlib, and Seaborn.

## Dataset

* Dataset Name: Iris Dataset
* Source: Seaborn Built-in Dataset (`sns.load_dataset('iris')`)
* Number of Records: 150
* Number of Features: 5

## Tools and Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

## Tasks Performed

1. Loaded the Iris dataset using Pandas and Seaborn.
2. Displayed the dataset structure using:

   * `.shape`
   * `.columns`
   * `.head()`
3. Created the following visualizations:

   * Scatter Plot
   * Histogram
   * Box Plot
4. Performed basic data inspection and summarization.

## Key Findings

* The dataset contains measurements of three species of Iris flowers: Setosa, Versicolor, and Virginica.
* Scatter plots show relationships between flower measurements.
* Histograms help understand the distribution of features.
* Box plots help identify the spread of data and potential outliers.

## Conclusion

This task provided hands-on experience with loading, exploring, and visualizing data using Python. It helped develop foundational skills in data analysis and exploratory data visualization.
